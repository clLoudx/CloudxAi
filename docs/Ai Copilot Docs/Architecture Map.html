4. Auto-Upgrade & Shadow Project System

LIVE SYSTEM
   │ (metrics, errors, usage)
   ▼
ADMIN AI (Governance)
   │ propose improvement
   ▼
SHADOW PROJECT COPY (Isolated)
   │ run STEP 1 → 10
   ▼
VALIDATION & COMPARISON
   ├─ PASS → Promote to Live
   └─ FAIL → Destroy Shadow

No live system is ever modified directly.

Auto-upgrade without isolation is forbidden.

5. Admin AI — Invisible Authority

6. User vs Admin Separation

- Auditable & logged
