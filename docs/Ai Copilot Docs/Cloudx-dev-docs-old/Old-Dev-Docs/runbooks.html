Runbooks & Alerting

Alerts

  • AIWorkerDown
  • AIQueueHigh
  • AIDLQDepthHigh

AIWorkerDown

Condition: up{job="ai-agent"} == 0 for 2m
Actions:
1) Check systemctl status aiagent-app and aiagent-rq
2) Inspect logs: sudo journalctl -u aiagent-app -n 200
3) Restart services or roll back deploy

AIQueueHigh

Condition: ai_queue_tasks > 50 for 1m
Actions:
1) Scale workers: docker compose up -d --scale worker=3
2) Investigate slow external calls in ai/adapter
3) Requeue/Inspect DLQ items

DLQ handling

Inspect DLQ:
redis-cli LRANGE ai_tasks_dlq 0 50
Requeue:
redis-cli RPOPLPUSH ai_tasks_dlq ai_tasks
Integrate runbooks into PagerDuty / Slack for automated notifications.
