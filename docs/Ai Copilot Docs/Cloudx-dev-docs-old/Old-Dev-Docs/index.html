Welcome

This website packages the full reverse-engineered project, the installer, monitoring, and a developer playbook. It is designed to be a canonical single-file website you can host or browse locally.

Use the left navigation to jump to focused sections. Each page contains copy/paste commands and exact file paths for deployment.

Getting started fast

  1. Download and unzip the project to your host.
  2. Review /etc/ai-agent/env and populate secrets.
  3. Run the full installer: sudo ./aiagent_full_power_installer.sh install (staging first).
  4. Verify services and metrics, then enable TLS if domain available.
