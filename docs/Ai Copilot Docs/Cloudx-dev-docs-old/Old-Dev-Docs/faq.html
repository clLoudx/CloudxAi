FAQ

Common

  • How to add a new agent?
  • How to change models?

How to add a new agent?

Add an entry in ai/prompt_templates/<agent>.json and expose CRUD endpoints in /api/agents. Persist agents in agents.json or DB.

How to change models per agent?

Set model field in agent config and the adapter will pick it up. Implement provider selection in ai/adapter.

For further questions, refer to Developer Guide.
