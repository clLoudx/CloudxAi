Location & Purpose

The installer script lives at project root: aiagent_full_power_installer.sh. It automates install, repair, venv setup, systemd unit creation, nginx template, and more.

Commands

sudo ./aiagent_full_power_installer.sh install
sudo ./aiagent_full_power_installer.sh repair
sudo ./aiagent_full_power_installer.sh apply_patch
sudo ./aiagent_full_power_installer.sh build_images
sudo ./aiagent_full_power_installer.sh export_images

Edit /etc/ai-agent/env before starting services.

File paths created

/opt/aiagent/           # project copied here by installer
/etc/ai-agent/env       # secrets (600)
 /etc/systemd/system/aiagent-app.service
 /etc/systemd/system/aiagent-rq.service
/etc/nginx/sites-available/ai-agent.conf
/var/backups/aiagent_*  # backups created during install
Always test installer in staging first.
